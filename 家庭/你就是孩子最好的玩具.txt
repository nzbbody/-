/**
@verbatim
1、转折点概念，当孩子在某一领域发生飞跃的进步，往往会在其他方面退步。比如会走路的前一天，可能会整整一夜不睡。
2、宝宝彻夜不眠，有可能有惊喜在早上等着你。
@endverbatim
*/
不眠之夜;

/**
@verbatim
1、孩子不善于调节自己的情绪，因此需要一个过渡性的物品来安慰他们。
2、孩子意识到有时候会和父母分开，开始建立独立性。就有可能对某物品产生依赖，安抚毯就是父母或者照顾者的象征，
	二者都可以提供平静和喜乐。
@endverbatim
*/
安抚毯具有象征意义;

/**
@verbatim
1、控制型父母与放任型父母，都不会教会孩子如何尊重他人的需要和体会他人的感受。
2、不要让孩子出于对你的恐惧而听从你的命令，而是应该出于对你的爱戴和尊重而听从你的建议。
3、如果家庭不能给孩子带来归属感和安全感，孩子就会到家以外的地方去寻找。
4、正确的做法是：采用折中的方法，跟孩子沟通和交流。
@endverbatim
*/
误区一：控制型父母与放任型父母;


/**
@verbatim
1、孩子跌倒，不要说，不疼不疼。这会让孩子心里很难受。
2、父母很容易把自己的观点和感受凌驾于孩子之上，应该站在孩子的角度思考问题，采用同情和理解的态度进行沟通。
3、轻视、低估和否定孩子的感受，会导致孩子缺乏同情心，并以自我为中心。
4、孩子难过的时候需要成年人的安慰，无论对社会还是他人的关怀，孩子最早都是从家庭中学到的。
	我们怎样对待孩子，孩子就会怎样对待别人。
@endverbatim
*/
误区二：轻视、低估和否定孩子的感受;


/**
@verbatim
1、用奖励来操纵孩子的表现是错误的。
2、孩子最希望得到的不是什么玩具，而是你的关注。
@endverbatim
*/
误区三：采用外部激励和奖赏的手段;


/**
@verbatim
1、什么是消极后果？
	孩子不听话，回来太晚，导致没饭吃饿肚子，父母很生气，说今天不能看电视。这就是消极后果。
2、孩子会困惑，为什么回来晚了，不能看电视。更好的策略是：
	对不起，你回来晚了，妈妈也想给你吃的，可是没有了。我们明天早上再吃，好吗？
	下次不要再回来晚了，好吗？
3、不要对孩子抱有不切实际的期望，父母的很多要求会超出小家伙的能力。
4、包容孩子是种乐趣，做出符合现实的期望，照顾孩子就会是很享受的事情。
@endverbatim
*/
误区四：以消极后果作为惩罚;


/**
@verbatim
1、播下种子
	提前告诉孩子在特定的场景下会发生什么事，应该怎么做，只告诉孩子让他乖一点，孩子是不知道怎么做的。
2、观察和判断
	观察和判断的本质是关注，孩子会因为你的关注，去做你期望他做的事情，并且在孩子需要的时候提供帮助。
3、聆听
	蹲下来和孩子的视线齐平，以便更好的眼神交流。
4、体察并理解孩子的感受
	承认并接受孩子的情绪，鼓励孩子把自己的感受说出来，而不是把情绪发泄到其他人身上。
5、同情
	当孩子感受到有人理解他的时候，孩子就更愿意与人交流。
@endverbatim
*/
情感引导的关键步骤;
